# Database Layer - SchoolSpace

এই ফোল্ডারে database schema, seed data, এবং migration সম্পর্কিত সব ফাইল আছে।

## 📁 ফাইল স্ট্রাকচার

```
database/
├── schema.prisma   ← Prisma ORM database schema (all models)
├── seed.ts         ← Dev database seeder (fake data for testing)
├── migrations/     ← Auto-generated by Prisma migrate (do not edit)
└── README.md       ← এই ফাইল
```

## 🗄️ Database Models

| Model | Description |
|-------|-------------|
| `User` | সকল ব্যবহারকারী (Admin, Teacher, Student) |
| `Student` | Student-specific profile (User এর extension) |
| `Teacher` | Teacher-specific profile (User এর extension) |
| `Class` | Class (e.g. Class 10) |
| `Section` | Class এর section (e.g. A, B, C) |
| `Subject` | প্রতিটি section এর subject (e.g. Math, Physics) |
| `Attendance` | প্রতিদিনের attendance record |
| `Mark` | Exam marks (Class Test, Mid, Final) |
| `Assignment` | Teacher-created assignments |
| `Submission` | Student assignment submissions |
| `Notice` | School notice board |
| `Message` | Teacher-Student messages |

## ⚙️ Setup কমান্ডগুলো

```bash
# প্রথমবার database তৈরি করতে (migration run):
npx prisma migrate dev --name init --schema=database/schema.prisma

# Database reset করতে (সব data মুছে নতুন করতে):
npx prisma migrate reset --schema=database/schema.prisma

# Seed data যোগ করতে:
npx tsx database/seed.ts

# Prisma Studio (visual DB browser) চালাতে:
npx prisma studio --schema=database/schema.prisma

# Client generate করতে:
npx prisma generate --schema=database/schema.prisma
```
